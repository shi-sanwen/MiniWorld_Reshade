# 迷你世界reshade安装工具（MiniReshade）

![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-green.svg)

嘿！欢迎来到迷你世界reshade安装工具的开源项目！本项目由一名初三学生开发，采用win32API构建UI，我们开始吧！╰(*°▽°*)╯

## 前言
**迷你世界光影包安装器** 是一个为《迷你世界》游戏设计的辅助工具，旨在帮助玩家轻松安装和管理光影包。通过本工具，用户可以快速检测游戏版本、下载预设光影包，并完成安装过程。

这是一个用于安装迷你世界游戏光影包的 Windows 应用程序。它支持多种版本的迷你世界，并提供一键安装光影包的功能。

## 功能特性

- **自动检测游戏版本**：自动扫描系统中安装的迷你世界游戏版本
- **多种版本支持**：支持官方版、4399版、7k7k版、国际版等多种版本
- **预设下载**：可下载由 YiRixc 和 天星大蛇 制作的光影预设
- **一键安装**：简化安装过程，只需点击一次即可完成光影包安装
- **注册表处理**：自动修改并导入注册表文件以确保光影包正常工作
- **现代化界面**：采用现代化的 UI 设计，提供良好的用户体验
- **扫描功能**：如果无法自动检测到游戏，可进行全盘扫描查找游戏安装目录

## 系统要求

- Windows 7 或更高版本
- 管理员权限（用于修改注册表）

## 安装说明

### 下载安装器

1. 访问项目的 [Releases](https://github.com/shisanwen-gmail/MiniWorld_Reshade/releases) 页面
2. 下载最新版本的安装器（.exe 文件）

### 运行程序

1. 右键点击下载的 `.exe` 文件
2. 运行`.exe`文件
3. 选择"以管理员身份运行"
4. 如果出现用户账户控制(UAC)提示，请点击"是"

## 使用教程

### 1. 启动程序

程序启动后会自动检测系统中安装的迷你世界游戏版本。如果检测成功，会在"检测到的游戏版本"区域显示找到的版本。

### 2. 添加自定义路径（可选）

如果程序未能自动检测到游戏安装目录：

1. 点击"添加自定义安装"按钮
2. 在弹出的文件浏览器中选择游戏的安装目录
3. 点击"确定"

### 3. 下载预设（可选）

1. 在"预设下载选项"区域勾选需要的预设：
   - "预设制作：YiRixc"
   - "预设制作：天星大蛇"
2. 点击"下载预设"按钮
3. 等待下载和解压完成

### 4. 安装光影包

1. 在"检测到的游戏版本"区域选择要安装光影包的游戏版本
2. 点击"安装"按钮
3. 程序会自动执行以下操作：
   - 检查并终止正在运行的游戏进程
   - 解压光影包文件到游戏目录
   - 修改注册表文件
   - 导入注册表文件到系统
4. 安装完成后会弹出提示框

### 5. 扫描功能

如果程序未能检测到游戏安装目录：

1. 点击"开始扫描"按钮
2. 程序会进行全盘扫描寻找游戏
3. 扫描过程中可以点击"暂停扫描"或"停止扫描"
4. 扫描完成后会自动更新游戏版本列表

## 界面说明

程序具有现代化的用户界面，包含以下几个主要部分：

- **版权声明区域**：显示软件的版权声明信息
- **更新日志**：显示软件的更新历史
- **检测到的游戏版本**：显示自动检测到的游戏版本列表
- **预设下载选项**：提供预设下载的选择
- **扫描控制按钮**：控制扫描过程（开始、暂停、停止）
- **安装按钮**：执行光影包安装操作
- **日志输出窗口**：显示操作过程中的详细日志信息
- **状态栏**：显示当前操作状态

## 技术实现

该程序使用 C++ 和 Windows API 开发，主要技术特点包括：

- 使用 GDI+ 实现现代化界面绘制
- 多线程处理扫描和安装任务
- 使用 7-Zip 解压工具处理压缩包
- 通过注册表操作实现光影包的正确配置
- 动画效果增强用户体验
- 自定义控件实现美观的界面元素

### 核心功能模块

1. **游戏检测模块**：
   - 通过注册表查询官方版本
   - 扫描用户目录查找其他版本
   - 全盘扫描功能作为备选方案

2. **安装模块**：
   - 解压光影包文件
   - 修改注册表文件
   - 导入注册表到系统

3. **UI模块**：
   - 自定义按钮、复选框、进度条等控件
   - 动画效果和现代化视觉设计
   - 响应式布局

## 注意事项

1. 程序需要以管理员权限运行才能正确修改注册表
2. 安装前会检查并尝试终止正在运行的游戏进程
3. 如果安装失败，程序提供重启后重试的选项
4. 请确保游戏未在运行状态下进行安装
5. 安装完成后建议重启游戏以确保效果生效
